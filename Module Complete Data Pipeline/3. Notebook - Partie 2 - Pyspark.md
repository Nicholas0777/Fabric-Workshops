# Notebook : Traitement de données des fichiers .json via PySpark
Le traitement des fichiers JSON peut être rendu complexe selon l'outil utilisé.<br>
En effet, bien que le JSON soit un format populaire et largement utilisé en raison de sa lisibilité pour les humains et de sa flexibilité pour représenter des données complexes, son traitement peut être particulier en raison de sa structure flexible, de l'absence de schéma rigide.

```json

    "site": "Machu Picchu",

}
```


## 1.2. Exemple sur l'ensemble des fichiers .json
1. Remplacer *WHC_1978* par **\*** afin de lire tous les fichiers **.json** dans le dossier et éxécuter encore la cellule:<br>
    ``` python
    df = spark.read.option("multiline", "true").json("Files/World Heritage Committee/*.json")
    # df now is a Spark DataFrame containing JSON data from "Files/World Heritage Committee/WHC_1978.json".
    display(df)
    ```

2. On va applatir les éléments de **Informations complementaires**
    ``` python
    df_new = df.select("ID", "Site", "Description", "`Informations complementaires`.*")
    display(df_new)
    ```
    
3. Pour garantir que les valeurs manquantes (NaN) ne soient pas interprétées comme des chiffres lors de la manipulation des données, même si la colonne est de type chaîne de caractères, nous devons remplacer les valeurs NaN par None. Cela peut être réalisé avec la méthode `replace` :
    ```python
    import numpy as np
    df_new = df_new.replace(np.nan, None)
    ```
