# Dataflow Gen2 : Présentation


## Différence entre Dataflow gen1 et Dataflow gen2
Le dataflow gen2 se distingue du dataflow classique de Power BI sur plusieurs aspects :
- Possibilité de sauvegarder le code sans exécuter
- Possibilité de sauvegarder les données de sorties dans un Lakehouse ou un Warehouse

## Notions à avoir
Si vous êtes déjà familier avec l'utilisation de Power Query (via Power BI ou Excel) les dataflows gen2 de MS Fabric devraient être faciles à appréhender.
- Utilisation de Power Query

# 1. [Pratique] Configurations du dataflow gen2
## 1.1. Créer un dataflow gen2 Fabric
1. Ouvrir votre workspace 
2. Si vous n'y êtes pas déjà, sélectionner dans l'expérience **Data Engineering**
3. Aller dans **New** > **Dataflow gen2**. Après quelques secondes, vous serez sur l'interface ci-dessous.<br>
    <img src="https://github.com/Nicholas0777/Fabric-Workshops/blob/main/Module%20Complete%20Data%20Pipeline/images/Dataflow-Gen2/Dataflow-gen2-create-a-dataflow-gen2.png" alt="" width="100%"/> <br><br>

## 1.2. Connecter le dataflow au Lakehouse Bronze
    
<img src="https://github.com/Nicholas0777/Fabric-Workshops/blob/main/Module%20Complete%20Data%20Pipeline/images/Dataflow-Gen2/Dataflow-Gen2-Load-data-from-Lakehouse.gif" alt="" width="80%"/> <br>

1. Aller dans **Get data** > **More**
2. La nouvelle fenêtre qui s'ouvre propose de sélectionner une nouvelle source de données. Au niveau de la barre de recherche entrer *"Lakehouse"* et sélectionner cette source :
    <img src="https://github.com/Nicholas0777/Fabric-Workshops/blob/main/Module%20Complete%20Data%20Pipeline/images/Dataflow-Gen2/Dataflow-gen2-Search-Lakehouse-Source.png" alt="" width="70%"/>
3. Appuyer sur **Next** 
4. A présent, chercher le workspace dans lequel se trouve le Lakehouse à connecter.<br>
    Déplier en cliquant sur **&#9655;** à côté du nom de votre workspace puis sur **&#9655;** à côté du nom de votre Lakehouse Bronze (Ici **DE_LH_Bronze**).
    Sélectionner le dossier **File** puis cliquer sur **Create**<br>
    <img src="https://github.com/Nicholas0777/Fabric-Workshops/blob/main/Module%20Complete%20Data%20Pipeline/images/Dataflow-Gen2/Dataflow-gen2-Select-Lakehouse-Files-Folder.png" alt="" width="70%"/> <br><br>
5. Après quelques secondes, vous obtiendrez ce résultat :
    <img src="https://github.com/Nicholas0777/Fabric-Workshops/blob/main/Module%20Complete%20Data%20Pipeline/images/Dataflow-Gen2/Dataflow-gen2-Interface-Components.png" alt="" width="70%"/> <br>
