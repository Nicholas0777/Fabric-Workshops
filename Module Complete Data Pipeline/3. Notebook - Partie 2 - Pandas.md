## 1.2. Création du dataframe complet
Vous avez compris comment :
- lire un fichier .json
- utiliser la méthode `json_normalize` pour aplatir les données imbriquées afin de transformer les données JSON en dataframe

1. Ici, nous allons utiliser une boucle pour répéter l'opération et ensuite combiner les résultats en un seul dataframe.
    ``` python
    racine_dossier_Lakehouse = "/lakehouse/default/" +"Files/World Heritage Committee"
    lst = []
    #for var_annee in ["1978","1979"]:
    for var_annee in range(1978,2024):
        print(var_annee)
        try:
            f = open(f"{racine_dossier_Lakehouse}/WHC_{var_annee}.json","r", encoding='utf-8')
            content = json.load(f)
            sub_df = pd.json_normalize(content)
            sub_df["Source"] = f"WHC_{var_annee}.json" # On ajoute une colonne pour indiquer le fichier source
            lst.append(sub_df)
        except:
            print(f"Pas de fichier pour l'année {var_annee}")

    df = pd.concat(lst).reset_index(drop=True)
    display( df.info() )
    df.head()
    ```

2. Pour garantir que les valeurs manquantes (NaN) ne soient pas interprétées comme des chiffres lors de la manipulation des données, même si la colonne est de type chaîne de caractères, nous devons remplacer les valeurs NaN par None. Cela peut être réalisé avec la méthode `replace` :
    ```python
    import numpy as np
    df = df.replace(np.nan, None)
    ```
### Version alternative
On utilise la méthode `listdir` pour obtenir la liste des fichiers du dossier **World Heritage Committee** (vu que **TOUS** les fichiers dans ce dossier sont similaires)
``` python
import os
import json
import pandas as pd
import numpy as np

racine_dossier_Lakehouse = "/lakehouse/default/" +"Files/World Heritage Committee"
folder_content = os.listdir(racine_dossier_Lakehouse) # Liste des fichiers dans le dossier
lst = []

for json_file in folder_content:
    print(json_file)
    f = open(f"{racine_dossier_Lakehouse}/{json_file}","r", encoding='utf-8')
    content = json.load(f)
    sub_df = pd.json_normalize(content)
    sub_df["Source"] = json_file # On ajoute une colonne pour indiquer le fichier source
    lst.append(sub_df)

df = pd.concat(lst).reset_index(drop=True).replace(np.nan, None)
display( df.info() )
df.head()
```
