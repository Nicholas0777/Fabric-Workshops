# 1. [Pratique] Importer le fichier Pays.csv dans le notebook avec Pandas
1. Dans le menu **...**, sélectionner pour **Pays.csv**, sélectionnez **Charger les données** > **Pandas**. Une nouvelle cellule de code contenant le code suivant devrait être ajoutée au notebook :<br>
    ``` python
    import pandas as pd
    # Load data into pandas DataFrame from "/lakehouse/default/" + "Files/Pays.csv"
    df = pd.read_csv("/lakehouse/default/" + "Files/Pays.csv",)
    display(df)
    ```
2. Cliquer sur le bouton **&#9655;** à gauche de la cellule pour l'éxcuter  (**Raccourci :** Vous pouvez également faire **CTRL** + **Entrée**)<br>
    Vous aurez une erreur d'exécution `UnicodeDecodeError: 'utf-8' codec can't decode byte 0xe9 in position 51: invalid continuation byte`.
3.  Par défaut, quand on ne le précise pas `read_csv` utilise l'encodage **UTF-8**. Notre fichier étant encodé en **ISO-8859-1**, il faudra donc ajouter dans `load` le paramètre `encoding='ISO-8859-1'`
    ``` python
    import pandas as pd
    # Load data into pandas DataFrame from "/lakehouse/default/" + "Files/Pays.csv"
    df = pd.read_csv("/lakehouse/default/" + "Files/Pays.csv", encoding='ISO-8859-1')
    display(df)
    ```
    <img src="https://github.com/Expertime/exp-data-fabric-workshops/blob/main/Images/Notebook/Notebook-Pays-csv-pandas.png" alt="" width="80%"/> <br>
    
4. Plus d'erreur d'exécution mais la séparation des lignes en plusieurs colonnes n'a pas été faite. Lorsqu'on ne le précise pas, le séparateur par défaut est la virgule `,`. Dans `load` on préciser le format des délimiteurs via `sep=";"`
    ``` python
    import pandas as pd
    # Load data into pandas DataFrame from "/lakehouse/default/" + "Files/Pays.csv"
    df = pd.read_csv("/lakehouse/default/" + "Files/Pays.csv", encoding='ISO-8859-1', sep=";", keep_default_na=False)
    display(df)
    # "keep_default_na=False" --> Pour eviter que Pandas remplace les valeurs "NA" en vide. Si l'option est en True, la "Namibie" qui a un code pays = NA aura sa valeur remplacee par du vide
    ```
    Le format est à présent satisfaisant

6. Avant de sauvegarder le dataframe dans le Lakehouse (comme table delta), il faut convertir ce dataframe **Pandas** en dataframe **Spark** et s'assurer que le type des colonnes est le bon.
    ``` python
    spark_df = spark.createDataFrame(df)
    spark_df.printSchema()
    ```
    _Output :_

    |-- region: string (nullable = true)

7. Sauvegarder le dataframe dans le **Lakehouse Silver**
    ``` python
    spark_df = spark_df.toDF( *(column.replace(' ', '_') for column in spark_df.columns) ) # On retire les espaces des noms de colonnes
    spark_df.write.format("delta").mode("overwrite").option('encoding','ISO-8859-1').option("nullValue", "\\N").saveAsTable("DE_LH_Silver.pays")
    # option('encoding','ISO-8859-1') = On precise encore l'encodage d'ecriture des donnees dans le lakehouse
    # option("nullValue", "\\N") = Sert a indiquer que les valeurs nulles sont egale a "\N" et non "NA" (Rappel : Code pays de Namibie = "NA")
    ```
    Une fois le code exécuté, aller dans le **Lakehouse Silver** afin de vérifier que la table a été créée.<br>

4. Dans la colonne **Catégorie**, selon que le site soit le suffixe *_danger* vient indique que le site est en péril. Pour continuer notre travail, nous allons dans un premier temps créer une nouvelle colonne contenant cette information puis nous allons uniformiser les valeurs la colonne **Catégorie** en faisant un remplacement de valeurs. <br>
Pour ajouter la colonne **En péril** dans le dataframe, qui indique si un site est en péril ou non, nous allons utiliser la ligne de code suivante:

    ``` python
    # Executer cette ligne de code pour verifier le resultat : df['Catégorie'].str.contains('danger'),"En péril"

    df.loc[~df['Catégorie'].str.contains('danger'),"En péril"] = False

    df.head()
    ```

7. Nous allons créer un nouveau DataFrame réordonné tout en conservant une copie de l'original afin de pouvoir revenir en arrière si nécessaire. Pour ce faire, nous allons utiliser le code suivant :
    ``` python
    df_reorder = df[['ID',  'Site', 'Catégorie', 'Site délisté', 'En péril', 'Pays', 'URL']] # nouveau dataframe
    df_reorder.columns = [x.replace("é","e").lower() for x in df_reorder.columns.tolist()]
    df_reorder.head()
    ```
    > **Remarques :** <br>
    Bien que Spark accepte les noms de colonnes commençant par une majuscule, il est généralement préférable de suivre une convention de nommage en minuscules avec des underscores pour améliorer la lisibilité et éviter les problèmes potentiels avec d'autres outils ou systèmes.<br>
    Par contre, Spark n'accepte pas de sauvegarder des tables qui ont des noms de colonnes ayant des caractères spéciaux ou des espaces. Il est donc important de s'assurer que les noms de colonnes sont conformes à ces restrictions lors de la manipulation de données avec Spark. <br><br>
    Pour généraliser le code on aurait pu utilier les **Regex** ou la bibliothèque `unicode` afin de remplacer les lettres qui ont des accents par des lettres sans accents.
8. Avant de sauvegarder le dataframe dans le Lakehouse (comme table delta), il faut convertir ce dataframe **Pandas** en dataframe **Spark**.
    Avant de sauvegarder le dataframe dans le Lakehouse (comme table delta), on va vérifier que le bon type est associé à chacune des colonnes.
    ``` python
    spark_df = spark.createDataFrame(df_reorder)
    spark_df.printSchema()
    ```
    _Output :_
    ```
    root
    |-- id: long (nullable = true)
    |-- site: string (nullable = true)
    |-- categorie: string (nullable = true)
    |-- site deliste: boolean (nullable = true)
    |-- en peril: boolean (nullable = false)
    |-- pays: string (nullable = true)
    |-- url: string (nullable = true)
    ```

    Le type de **id** n'est pas le bon, on va alors préciser le bon type avec la ligne de code suivante :
    ```python
    from pyspark.sql.types import IntegerType, StringType, BooleanType
    spark_df = df_reorder.withColumn("id",col("id").cast(IntegerType())) \
                    .withColumn("site",col("site").cast(StringType())) \
                    .withColumn("categorie",col("categorie").cast(StringType())) \
                    .withColumn("site deliste",col("site deliste").cast(BooleanType())) \
                    .withColumn("en peril",col("en peril").cast(BooleanType())) \
                    .withColumn("pays",col("pays").cast(StringType())) \
                    .withColumn("url",col("url").cast(StringType())) \
                
    spark_df.printSchema()
    ```
    _Output :_
    ```
    root
    |-- id: integer (nullable = true)
    |-- site: string (nullable = true)
    |-- categorie: string (nullable = true)
    |-- site deliste: boolean (nullable = true)
    |-- en peril: boolean (nullable = false)
    |-- pays: string (nullable = true)
    |-- url: string (nullable = true)
    ```
    > **Remarque :**<br> Les types `long` et `integer` représentent tous deux des entiers, la différence majeure réside dans le nombre de bytes. Avec `integer`, on est en 4-bytes et les valeurs peuvent aller de -2,147,483,648 à 2,147,483,647. En `long` on est sur 8-bytes et les valeurs vont de -9,223,372,036,854,775,808 à 9,223,372,036,854,775,807.<br>
    Pour notre cas, le type `integer` est suffisant.

## Python : Exemples de méthodes associées à des type de données
| **Type de donnée** | **Méthode**          | **Description**                                                                         | **Exemple**                                                   |
|--------------------|----------------------|-----------------------------------------------------------------------------------------|---------------------------------------------------------------|
| **str**            | `str.lower()`        | Convertit tous les caractères en minuscules                                             | `"Bonjour".lower() -> "bonjour"`                              |
| **str**            | `str.upper()`        | Convertit tous les caractères en majuscules                                             | `"Bonjour".upper() -> "BONJOUR"`                              |
| **str**            | `str.replace(old, new)` | Remplace toutes les occurrences de `old` par `new`                                      | `"Bonjour".replace("jour", "soir") -> "Bonsoir"`             |
| **str**            | `str.split(sep)`     | Divise la chaîne en une liste de sous-chaînes en utilisant `sep` comme séparateur       | `"Bonjour tout le monde".split(" ") -> ['Bonjour', 'tout', 'le', 'monde']` |
| **str**            | `str.strip()`        | Supprime les espaces au début et à la fin de la chaîne                                  | `"  Bonjour  ".strip() -> "Bonjour"`                          |
| **list**           | `list.append(x)`     | Ajoute l'élément `x` à la fin de la liste                                               | `[1, 2, 3].append(4) -> [1, 2, 3, 4]`                         |
| **list**           | `list.remove(x)`     | Supprime la première occurrence de l'élément `x` de la liste                            | `[1, 2, 3, 2].remove(2) -> [1, 3, 2]`                         |
| **list**           | `list.pop(i)`        | Supprime l'élément à la position `i` et le renvoie                                      | `[1, 2, 3].pop(1) -> 2, [1, 3]`                               |
| **list**           | `list.sort()`        | Trie la liste en place                                                                  | `[3, 1, 2].sort() -> [1, 2, 3]`                               |
| **list**           | `list.reverse()`     | Renverse l'ordre des éléments dans la liste                                             | `[1, 2, 3].reverse() -> [3, 2, 1]`                            |
| **dict**           | `dict.keys()`        | Renvoie une vue des clés du dictionnaire                                                | `{"a": 1, "b": 2}.keys() -> dict_keys(['a', 'b'])`            |
| **dict**           | `dict.values()`      | Renvoie une vue des valeurs du dictionnaire                                             | `{"a": 1, "b": 2}.values() -> dict_values([1, 2])`            |
| **dict**           | `dict.items()`       | Renvoie une vue des paires clé-valeur du dictionnaire                                   | `{"a": 1, "b": 2}.items() -> dict_items([('a', 1), ('b', 2)])`|
| **dict**           | `dict.get(key)`      | Renvoie la valeur associée à `key`, ou `None` si `key` n'existe pas                     | `{"a": 1}.get("a") -> 1, {"a": 1}.get("b") -> None`           |
| **pandas.DataFrame** | `DataFrame.head(n)` | Renvoie les `n` premières lignes du DataFrame                                           | `df.head(5)`                                                  |
| **pandas.DataFrame** | `DataFrame.tail(n)` | Renvoie les `n` dernières lignes du DataFrame                                           | `df.tail(5)`                                                  |
| **pandas.DataFrame** | `DataFrame.describe()` | Renvoie les statistiques descriptives du DataFrame                                      | `df.describe()`                                               |
| **pandas.DataFrame** | `DataFrame.drop(columns)` | Supprime les colonnes spécifiées du DataFrame                                           | `df.drop(['col1', 'col2'], axis=1)`                           |
| **pandas.DataFrame** | `DataFrame.loc[]` | Accède à un groupe de lignes et de colonnes par des étiquettes ou une condition booléenne | `df.loc[df['col'] > 0]`                                       |
| **numpy.ndarray**     | `ndarray.shape`           | Renvoie les dimensions du tableau                                                       | `arr.shape`                                                   |
| **numpy.ndarray**     | `ndarray.size`            | Renvoie le nombre total d'éléments dans le tableau                                      | `arr.size`                                                    |
| **numpy.ndarray**     | `ndarray.dtype`           | Renvoie le type des éléments dans le tableau                                            | `arr.dtype`                                                   |
| **numpy.ndarray**     | `ndarray.reshape(newshape)`| Change la forme du tableau sans en modifier les données                                 | `arr.reshape((2, 3))`                                         |
| **numpy.ndarray**     | `ndarray.flatten()`       | Aplatit le tableau en un tableau à une dimension                                        | `arr.flatten()`                                               |
| **numpy.ndarray**     | `ndarray.mean()`          | Calcule la moyenne des éléments du tableau                                              | `arr.mean()`                                                  |
| **numpy.ndarray**     | `ndarray.sum()`           | Calcule la somme des éléments du tableau                                                | `arr.sum()`                                                   |
| **numpy.ndarray**     | `ndarray.max()`           | Renvoie la valeur maximale des éléments du tableau                                      | `arr.max()`                                                   |
| **numpy.ndarray**     | `ndarray.min()`           | Renvoie la valeur minimale des éléments du tableau                                      | `arr.min()`                                                   |
| **numpy.ndarray**     | `ndarray.std()`           | Calcule l'écart type des éléments du tableau                                            | `arr.std()`                                                   |
