# Alimenter un Lakehouse
Pour alimenter un Lakehouse, Microsoft Fabric propose différentes méthodes en fonction des besoins :
- **Automatique :**
    - **Pipelines de données :** Idéal pour des processus de données récurrents et pour orchestrer des tâches complexes d'ETL (Extract, Transform, Load).
    - **Dataflows Gen2 :** Utiles pour des transformations de données No code ou Low code.
    - **Notebooks :** Pratique pour effectuer des analyses avancées ou des transformations de données complexes.
- **Manuel :**
    - **Shortcuts :** Effectuez des chargements manuels ponctuels.
    - **Dépôts manuels :** Pratique pour les utilisateurs qui souhaitent rapidement importer des fichiers de données sans configuration complexe.
- **Streaming en temps réel :**<br>
    Idéal pour les cas d’usage nécessitant une ingestion continue de données en temps réel.
- **Réplication de bases de données :**<br>
    Utile pour maintenir une synchronisation en temps quasi-réel entre des bases de données opérationnelles et le Lakehouse.

## 2. [PRATIQUE] Déposer manuellement les fichiers dans le Lakehouse Bronze
1. Télécharger le fichier **.zip** ([Lien](https://raw.githubusercontent.com/Expertime/exp-data-fabric-workshops/main/Input/demo.zip?token=GHSAT0AAAAAACU34QISURBTWUS3HKOMRQE4ZUVG24Q)) et le décompresser
2. Ouvrir le Lakehouse de la couche Bronze<br>
    Vous serez alors dans l'**interface explorateur** du Lakehouse. Depuis cette interface, il est possible de :
    - Charger ou explorer des fichiers ou des tables
    - Intégrer le lakehouse dans un notebook, un dataflow gen2, une pipeline de données, etc.<br>
3. Réaliser les actions suivantes<br>
    Dossier *Fichiers* > **...** > **Charger** > **Charger un dossier** > sélectionner le dossier *World Heritage Committee* > Importer > Charger <br>
    <img src="https://github.com/Nicholas0777/Fabric-Workshops/blob/main/Module%20Complete%20Data%20Pipeline/images/Lakehouse/Lakehouse_Ingestion_Dossier.gif" alt="" width="60%"/> <br>
    Le dossier *World Heritage Committee* contient la liste annuelle des retours du comité de patrimoine mondial
4. Réaliser la même action pour charger les fichiers (cette fois-ci :  Dossier *Fichiers* > **...** > **Charger** > **Charger un fichier** > *sélectionner le fichier adéquat* > Importer > Charger)
    - Liste_2023.xlsx
    - Pays.csv
5. Une fois les dépôt réalisés, votre Lakehouse devrait ressembler à ceci :<br>
    <img src="Images/Lakehouse/Lakehouse-Fichiers.png" alt="" width="50%"/> <br>
    Vous pouvez naviguer dans les fichiers afin d'avoir une première idée sur leur contenu.
