# Propositions de solutions
## Niveau 1
`Agrégations | Condition WHERE | Tri | Jointure`
1. Depuis sa création, combien de sites ont été inscrits dans la liste du patrimoine mondial de l'UNESCO ?
    ```sql
    SELECT COUNT(DISTINCT [id])
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites]
    ```

2. Combien de sites sont actuellement inscrit dans la liste ?
    ```sql
    SELECT COUNT(DISTINCT [id])
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites]
    WHERE [site_deliste] = 0
    ```

3. Quels sont les sites qui sont en péril ?
    ```sql
    SELECT DISTINCT [id], [site]
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites]
    WHERE [en_peril] = 1
    ```
4. Combien de sites naturels sont en péril ?
    ```sql
    SELECT DISTINCT [id], [site], [categorie]
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites]
    WHERE [categorie] = 'Naturel' AND [en_peril] = 1
    ```
5. Quel est le nombre de sites par pays ?
    ```sql
    SELECT [pays], COUNT([id]) AS 'nombre_sites'
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites]
    GROUP BY [pays]
    ORDER BY 'nombre_sites' DESC
    ```
6. Dans la région `Asie et Pacifique`, quels sont les 6 pays qui ont le plus de sites inscrits ? 
    ```sql
    SELECT 
        TOP(6) p.[code_pays], p.[pays], COUNT(whss.[id]) AS 'nombre_sites'
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites] whss
    INNER JOIN [DE_LH_Silver].[dbo].[pays] p
        ON p.[pays] = whss.[pays]
    WHERE p.[region] = 'Asie et Pacifique'
    GROUP BY p.[code_pays], p.[pays]
    ORDER BY 'nombre_sites' DESC
    ```

## Niveau 2
`CTE | Condition HAVING | Jointures | Sous-requêtes`
1. Quelle est la liste des pays sans site ?
    ```sql
    SELECT 
        rg.*
    FROM [DE_LH_Silver].[dbo].[ref_geo] rg
    LEFT JOIN [DE_LH_Silver].[dbo].[pays] p
        ON p.[code_pays] = rg.[iso2]
    WHERE p.[code_pays] IS NULL
    ```

2. Combien de pays ont au minimum 26 sites ?<br>
    ```sql
    SELECT 
        whss.[pays], COUNT(whss.[id]) AS 'nombre_sites'
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites] whss
    GROUP BY whss.[pays]
    HAVING COUNT(whss.[id]) >= 26
    ORDER BY 'nombre_sites' DESC
    ```

3. Liste des sites transfrontaliers (qui sont répartis sur plusieurs pays)
    ```sql
    SELECT 
        whss.[id], whss.[site], COUNT(whss.[pays]) AS 'nombre_de_pays'
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites] whss
    GROUP BY whss.[id], whss.[site]
    HAVING COUNT(whss.[pays]) > 1
    ORDER BY 'nombre_de_pays' DESC
    ```

4. En quelles années des pays ont eu au moins 5 sites inscrits la même année ? <br> 
    _Afficher le résultat sous forme d'une table contenant l'année, le pays et le nombre de sites qui ont été inscrits_
    ```sql
    SELECT 
        whsd.[date_dinscription] AS 'annee',
        whs.[pays] AS 'pays',
        COUNT(DISTINCT whsd.[id]) AS 'nombre_de_sites'
    FROM [DE_LH_Silver].[dbo].[world_heritage_sites_details] whsd
    JOIN [DE_LH_Silver].[dbo].[world_heritage_sites] whs
        ON whs.[id] = whsd.[id]
    GROUP BY whsd.[date_dinscription], whs.[pays]
    HAVING COUNT(DISTINCT whsd.[id]) >= 5
    ORDER BY whsd.[date_dinscription]
    ```

5. Quelle est l'année où il y a eu le plus de sites inscrits ? <br>
    ```sql
    WITH site_par_annee (annee, nombre_de_sites) AS(
        SELECT 
            whsd.[date_dinscription] AS 'annee',
            COUNT(DISTINCT [id]) AS 'nombre_de_sites'
        FROM [DE_LH_Silver].[dbo].[world_heritage_sites_details] whsd
        GROUP BY whsd.[date_dinscription]
        )
    SELECT annee
    FROM site_par_annee
    WHERE nombre_de_sites = (SELECT MAX(nombre_de_sites) FROM site_par_annee)
    ```

6. Quel est le pays qui a la plus grande superficie totale occupée par ses sites ? (exclure les sites transfrontaliers)

7. Par continent, est le ratio du nombre de pays sans sites par rapport au nombre total de pays ?<br>
    _Afficher le résultat sous forme de table contenant le continent, le nombre pays, le nombre de pays sans sites et le ratio_
    - **Version 1 :** Code utilisant les `CTEs`, voir **version 2** pour une version plus optimisée
        ```sql
        WITH pays_par_continent(continent,total_pays) AS
            (
                SELECT rg.[continent], COUNT(DISTINCT rg.[iso2]) AS 'total_pays'
                FROM [DE_LH_Silver].[dbo].[ref_geo] AS rg
                GROUP BY rg.[continent]
            ),
            pays_sans_site_par_continent(continent,total_pays_sans_site) AS
            (
                SELECT rg.[continent], COUNT(DISTINCT rg.[iso2]) AS 'total_pays_sans_site'
                FROM [DE_LH_Silver].[dbo].[ref_geo] AS rg
                LEFT JOIN [DE_LH_Silver].[dbo].[pays] AS p
                    ON p.[code_pays] = rg.[iso2]
                WHERE p.[code_pays] IS NULL
                GROUP BY rg.[continent]
            )
        SELECT 
            ppc.[continent], 
            [total_pays], 
            [total_pays_sans_site], 
            CAST( 100.0*[total_pays_sans_site]/[total_pays] AS DECIMAL(10,2)) AS 'ratio'
        FROM pays_par_continent AS ppc
        JOIN pays_sans_site_par_continent AS psspc
            ON psspc.[continent] = ppc.[continent]
        ORDER BY [continent]
        ```
    - **Version 2 :** On se sert de `CASE WHEN` pour vérifier si un pays a un site ou non.
        ```sql
        SELECT 
            rg.continent ,
            COUNT(rg.iso2) AS 'total_pays',
            SUM(CASE WHEN p.code_pays IS NULL THEN 1 ELSE 0 END) AS 'total_pays_sans_site',
            CAST( 100.0*SUM(CASE WHEN p.code_pays IS NULL THEN 1 ELSE 0 END)/COUNT(rg.iso2) AS DECIMAL(10,2) ) AS 'ratio'
        FROM [dbo].[ref_geo] AS rg
        LEFT JOIN [dbo].[pays] AS p
            ON p.code_pays = rg.iso2 
        GROUP BY rg.continent 
        ORDER BY rg.continent 
        ```

## Niveau 3
`Agrégations+ | CASE WHEN | CTE | Fonctions fenêtre | Sous-requêtes`
1. En quelle année la France a t'elle eu son 30ième culturel site inscrit ?
    ```sql
    WITH tmp_table(id, site, pays,date_dinscription,site_rank)  AS
        (
        SELECT 
            whs.[id], whs.[site], whs.[pays], whsd.[date_dinscription],
            RANK() OVER(PARTITION BY whs.[pays] ORDER BY whs.[id]) as 'site_rank'
        FROM [dbo].[world_heritage_sites] whs
        INNER JOIN [dbo].[world_heritage_sites_details] whsd
            ON whsd.[id] = whs.[id]
        WHERE whs.[categorie] = 'Culturel' and whs.[pays] = 'France'
        )
    SELECT [id], [site], [date_dinscription]
    FROM tmp_table
    WHERE [site_rank] = 30
    ```

2. Quel pays a été le premier à obtenir 10 sites culturels inscrits ?
    ```sql
    WITH tmp_table(id, site, pays,date_dinscription,site_rank)  AS
        (
        SELECT 
            whs.[id], whs.[site], whs.[pays], whsd.[date_dinscription],
            RANK() OVER(PARTITION BY whs.[pays] ORDER BY whs.[id]) as 'site_rank'
        FROM [dbo].[world_heritage_sites] whs
        INNER JOIN [dbo].[world_heritage_sites_details] whsd
            ON whsd.[id] = whs.[id]
        WHERE whs.[categorie] = 'Culturel'
        )
    SELECT *
    FROM tmp_table
    WHERE [site_rank] = 10 
        and [date_dinscription] = (SELECT MIN([date_dinscription]) FROM tmp_table WHERE [site_rank] = 10)
    ```

3. Quels sont les 10 premiers pays à avoir obtenu 12 sites inscrits ?
    ```sql
    WITH ranking_sites_par_pays(pays,date_dinscription,site_rank)  AS
        (
        SELECT 
            whs.[pays], whsd.[date_dinscription],
            RANK() OVER(PARTITION BY whs.[pays] ORDER BY whs.[id]) as 'site_rank'
            --par pays on tri les sites selon leur id
        FROM [dbo].[world_heritage_sites] whs
        INNER JOIN [dbo].[world_heritage_sites_details] whsd
            ON whsd.[id] = whs.[id]
        ),
        pays_top_12_sites(pays,date_dinscription,annee_rank) AS
        (
        SELECT [pays], [date_dinscription],
            DENSE_RANK() OVER(ORDER BY [date_dinscription]) as 'annee_rank' 
            --pays_classes_selon_lannee_ou_ils_ont_eu_leur_15eme_site_inscrit
        FROM ranking_sites_par_pays
        WHERE [site_rank]=12 -- on garde uniquement le 12e site inscrit
        )
    SELECT [pays], [date_dinscription], [annee_rank]
    FROM pays_top_12_sites
    WHERE [annee_rank]<=10 --les 10 premiers pays a avoir obtenu 12 sites inscrits
    ORDER BY [annee_rank], [date_dinscription]
    ```

    - Sous-table : `ranking_sites_par_pays`
        ```sql
        SELECT 
            whs.[id], whs.[site],whs.[pays], whsd.[date_dinscription],
            RANK() OVER(PARTITION BY whs.[pays] ORDER BY whs.[id]) as 'site_rank' 
            --par pays on tri les sites selon leur id
        FROM [dbo].[world_heritage_sites] whs
        INNER JOIN [dbo].[world_heritage_sites_details] whsd
            ON whsd.[id] = whs.[id]
        ORDER BY [pays], [site_rank]
        ```

5. Quelles régions ont au minimum 25% de sites naturels par rapport à la totalité de leurs sites ?
    ```sql
    WITH tmp_table(region, categorie, id) AS (
        SELECT DISTINCT p.[region], whs.[categorie], whs.[id]
        FROM [dbo].[world_heritage_sites] whs
        INNER JOIN [dbo].[pays] p
            ON whs.[pays] = p.[pays]
    )
    SELECT 
        [region],
        SUM(CASE WHEN [categorie] = 'Naturel' THEN 1 ELSE 0 END) AS 'nombre_de_sites_naturels',
        SUM(CASE WHEN [categorie] = 'Mixte' THEN 1 ELSE 0 END) AS 'nombre_de_sites_mixtes',
        SUM(CASE WHEN [categorie] = 'Culturel' THEN 1 ELSE 0 END) AS 'nombre_de_sites_culturels',
        COUNT([id]) AS 'total_sites',
        100.0*SUM(CASE WHEN [categorie] = 'Naturel' THEN 1 ELSE 0 END)/COUNT([id]) AS 'ratio'
    FROM tmp_table
    GROUP BY [region]
    HAVING 100.0*SUM(CASE WHEN [categorie] = 'Naturel' THEN 1 ELSE 0 END)/COUNT([id]) > 25.0
    ```

    WITH liste_sites_par_region_categorie(code_region, region, id, categorie) AS (
        SELECT 
            DISTINCT p.[code_region], p.[region], whs.[id], whs.[categorie]

        sites_par_region(code_region, region, culturel, naturel, mixte, total) AS (
        SELECT
